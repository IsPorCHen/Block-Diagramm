# Генератор блок-схем из Python кода

Веб-приложение для автоматической генерации блок-схем из исходного кода Python в соответствии с ГОСТ.

## Установка и запуск

### Требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Шаги установки

1. **Клонируйте или скачайте проект**
   ```bash
   git clone https://github.com/IsPorCHen/Diagramm.git
   cd Diagramm
   ```

2. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```
   
   Или на некоторых системах:
   ```bash
   pip install -r requirements.txt --break-system-packages
   ```

3. **Запустите приложение**
   ```bash
   python app.py 
   ```

4. **Откройте в браузере**
   ```
   http://localhost:5000
   ```

## Использование

### Загрузка файла

1. Нажмите кнопку "Выбрать файл" или перетащите файл .py в область загрузки
2. Убедитесь, что файл имеет расширение .py и размер не более 1 МБ
3. Нажмите "Сгенерировать блок-схему"

### Работа с блок-схемой

- **Масштабирование:**
  - Кнопка "+" - увеличить масштаб
  - Кнопка "−" - уменьшить масштаб
  - Кнопка "⟲" - сбросить масштаб

- **Экспорт:**
  - Нажмите "Экспортировать в PNG"
  - Файл автоматически загрузится на ваше устройство

### Просмотр кода

После генерации блок-схемы исходный код будет отображен в отдельной секции ниже.

## Примеры

В проекте включен пример файла `example_numbers.py` с функцией вычисления операций над числами.

### Пример простого кода:

```python
def check_number(x):
    if x > 0:
        print("Положительное")
    elif x < 0:
        print("Отрицательное")
    else:
        print("Ноль")

number = int(input("Введите число: "))
check_number(number)
```

## Ограничения

- Максимальный размер файла: **1 МБ**
- Поддерживаемый язык: **Python**
- Формат экспорта: **PNG**
- Время обработки: до 5 секунд для кода объемом 500 строк

## Требования к коду

Для корректной генерации блок-схемы код должен:
- Быть синтаксически корректным Python кодом
- Использовать стандартные конструкции (if/elif/else, for, while, def)
- Иметь правильную структуру и отступы

## Обработка ошибок

Приложение обрабатывает следующие ошибки:
- ❌ Некорректное расширение файла
- ❌ Превышение размера файла
- ❌ Синтаксические ошибки Python
- ❌ Ошибки генерации блок-схемы

При возникновении ошибки отображается информативное сообщение с описанием проблемы.
