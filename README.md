# <div align="center"><img src="https://github.com/IsPorCHen/Block-Diagramm/blob/main/logo.png?raw=true" alt="Flowchart Generator Logo" width="400" height="80"/></div>

* Flowchart Generator — это веб-приложение для автоматической генерации блок-схем из исходного кода на языке Python. Система анализирует загруженный код и создаёт визуальное представление алгоритма в соответствии с ГОСТ 19.701-90.
* Проект реализован с использованием стека технологий HTML, CSS, JavaScript, Python и фреймворка Flask.

![alt text](https://github.com/IsPorCHen/Block-Diagramm/blob/main/screenshots/main_interface.png "Главный интерфейс")

На сайте присутствует генератор блок-схем, работающий по принципу анализа синтаксического дерева кода Python.
![alt text](https://github.com/IsPorCHen/Block-Diagramm/blob/main/screenshots/flowchart_example.png "Пример блок-схемы")

# Технологический стек
* Фронтенд: HTML5, CSS3, Vanilla JavaScript
* Бэкенд: Python, Flask
* Анализ кода: модуль ast (Abstract Syntax Tree)
* Визуализация: SVG

# Описание работы генератора
* Пользователь загружает файл .py через веб-интерфейс (drag-and-drop или выбор файла).
* Система проверяет формат файла и валидирует синтаксис Python.
* Код анализируется с помощью модуля ast, строится абстрактное синтаксическое дерево.
* На основе AST формируется граф из узлов и рёбер.
* Граф преобразуется в SVG-изображение блок-схемы.
* Пользователь может масштабировать, перемещать и экспортировать блок-схему в PNG.

# Поддерживаемые конструкции Python
* Условные операторы: if, elif, else
* Циклы: for, while
* Функции: def с параметрами и возвращаемыми значениями
* Классы: class с полями и методами
* Обработка исключений: try, except, finally
* Операторы присваивания и вызовы функций

# Ключевые особенности
* Автоматическое распознавание всех основных конструкций языка Python.

* Интуитивно понятный интерфейс с поддержкой drag-and-drop загрузки файлов.

* Независимое управление каждой блок-схемой: масштабирование, перемещение, экспорт.

* Геометрические фигуры соответствуют ГОСТ 19.701-90: овал (терминатор), прямоугольник (процесс), ромб (условие), шестиугольник (цикл), параллелограмм (ввод/вывод).

* Корректная визуализация вложенных структур произвольной глубины.

# Требования для локального запуска
* Python 3.10+
* Flask 2.0+

# Установка и запуск локально
```bash
# Клонировать репозиторий
git clone https://github.com/IsPorCHen/Block-Diagramm.git
cd Block-Diagramm

# Установить зависимости
pip install -r requirements.txt

# Запустить сервер
python app.py
```
После запуска откройте браузер и перейдите по адресу: http://localhost:5000
